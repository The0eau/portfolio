February 2025 - Present

This project focuses on visualizing bike traffic across Boston using real-time data from various bike stations. It leverages Mapbox to create an interactive map displaying bike arrivals, departures, and station activity, offering insights into bike traffic patterns in the city.

Key Features

  • Interactive Map: Built using Mapbox, the map allows users to zoom in and explore bike station locations, viewing traffic patterns and station activity.
  • Real-time Data: The visualization updates dynamically, reflecting real-time traffic data such as bike arrivals and departures.
  • Custom SVG Elements: Custom icons and overlays highlight important data points, enhancing the map's readability and interactivity.
  • Optimized Performance: Optimized rendering techniques ensure a smooth experience, even with large datasets.

Technical Details

This project utilizes Mapbox for geospatial visualization, combined with JavaScript to handle data filtering and processing. D3.js is used to add dynamic SVG elements, such as bike station markers, with interactive hover and click features.

Hosting & Access

The project is hosted on GitHub Pages, providing easy access to the interactive map. You can explore bike traffic in Boston at any time through the live site.
