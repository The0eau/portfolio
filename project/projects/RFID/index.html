Project FPA: Autonomous Robot Creation for Competition Using RFID Technology

This project, part of an autonomous robot competition, involved developing a robot capable of interacting with its environment via RFID technology. The robot had to detect and react to RFID tags placed along its path while optimizing its autonomous navigation performance. The system was designed to operate fully autonomously without human intervention, with the goal of maximizing the robot's efficiency and speed on a predefined course.

Skills Applied

  • Autonomous Robot Design: Created and programmed the robot to navigate autonomously while detecting and reacting to RFID tags.
  • Microcontroller Programming: Developed firmware allowing the robot to interact with RFID tags and follow the course without external intervention.
  • RFID Technology Use: Integrated an RFID system so the robot could detect tags and make real-time decisions.
  • Performance Optimization: Improved the robot's speed, accuracy, and autonomy to make it more competitive.

Objectives

  • Full Autonomy: Enable the robot to navigate and react independently to RFID tags.
  • Navigation Optimization: Ensure smooth and precise navigation along the competition course.
  • High Performance: Improve the robot's speed and responsiveness for optimal performance.

Results Achieved

  • A 25% reduction in the robot's response time, thanks to optimized RFID sensor management and navigation algorithms.
  • A 50% improvement in the RFID tag detection range, allowing better consideration of the different elements on the course.
  • Positive feedback during competition demonstrations, with outstanding performance in the trials.

Key Skills

  • RFID Technology
  • Microcontroller Programming
  • Embedded System Optimization
