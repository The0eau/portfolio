SAR (Système Asynchrone)

The SAR project involved designing and validating an asynchronous processor using VHDL. It included several sub-components such as the register bank, functional units, address generator, and an automaton, all validated through detailed functional simulations.

Key Features

  • Processor Operational Unit: Simulation and synthesis of components (register bank, functional units, output register) using Flip-Flops and LUTs.
  • Register Bank: 72 Flip-Flops, functional validation during the activation of a load signal.
  • Functional Unit: 86 LUTs used for binary signals.
  • Address Generator and Automaton: Validation through functional simulation with optimized resources.

Results

  • Functional Simulation: Validated for all components.
  • Resource Optimization: Minimized resource usage (e.g., 10 LUTs, 16 registers for the automaton).
