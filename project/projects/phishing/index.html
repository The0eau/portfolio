Phishing Link Detector (AI)

This project focuses on developing an AI-based phishing link detector. The goal was to create a model capable of identifying phishing links using machine learning, by comparing different supervised models to determine the most effective one.

Key Features

  • Data Preparation and Analysis: Collected and cleaned datasets containing both phishing and legitimate links, followed by preprocessing for analysis and feature extraction.
  • Comparison of Supervised Models: Evaluated and compared several supervised machine learning models to identify the most effective one for phishing link detection.
  • Models Used:
    • Decision Trees
    • Ensemble Methods
    • Neural Networks
    • Support Vector Machines
  • Model Evaluation: Analyzed the performance of each model in terms of accuracy, precision, and recall, identifying the best model for detecting phishing links.
  • Interpretation & Report: Interpreted the results and created a detailed report summarizing the findings, including the best model's effectiveness in detecting phishing links.
