Decision Support Website for French Educational Institutions

This project involves the creation of a decision support website for French educational institutions. The platform aims to improve strategic financial guidance through data collection, analysis, statistical modeling, machine learning, and interactive data visualizations.

Key Features

  • Data Collection and Analysis: Gathered and processed data from various educational institutions in France, focusing on financial, academic, and operational metrics.
  • Statistical Modeling: Applied statistical methods to analyze the data and identify patterns that can inform financial and strategic decision-making.
  • Machine Learning: Incorporated machine learning algorithms to enhance predictions and recommendations based on historical data.
  • Interactive Data Visualization: Developed interactive visualizations to display key insights and trends, allowing users to explore the data dynamically.
  • Dashboard on a Website: Created a user-friendly dashboard hosted on a website, providing an accessible platform for stakeholders to access and interpret the data and insights.

Hosting & Access

The project is hosted on a website for easy access, allowing users to interact with the dashboard and view the results of the data analysis and machine learning models in real time.
