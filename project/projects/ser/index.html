Optimization of a Logistics Network

This project demonstrates my ability to solve optimization problems, a key skill for finance applications such as portfolio allocation and risk management. The focus of this project was on optimizing a logistics network using Mixed Integer Linear Programming (MILP), with tools such as Python (Pandas, Numpy, PULP) to model and solve the problem efficiently.

Key Features

  • Optimization (MILP): Applied Mixed Integer Linear Programming (MILP) techniques to optimize logistics network design, minimizing costs and improving efficiency while meeting all constraints.
  • Python (Pandas, Numpy, PULP): Utilized Python libraries such as Pandas and Numpy for data manipulation and analysis, and PULP for formulating and solving the optimization model.
  • Logistics Network Design: Modeled a logistics network, determining the optimal locations, capacities, and transportation routes to minimize operational costs and enhance performance.
  • Real-World Applications: Focused on solving real-world logistics challenges that have applications in supply chain management, finance, and operations research.
