Dynamic Visualizations

January 2025 - March 2025

Visualization: Link

This project involves the creation of interactive charts to visualize continuous glucose levels and physiological data using D3.js. The goal was to build dynamic, user-driven tools for real-time data exploration and analysis, offering insights into glucose trends and meal patterns.

Key Features

  • Interactive Charts: Developed charts using D3.js to visualize continuous glucose levels and physiological data, allowing users to interact with the data.
  • User-driven Exploration: Built dynamic tools for real-time exploration, enabling users to analyze glucose trends, meal patterns, and their relationships.
  • Customized Visualizations: Created tailored visualizations to highlight key insights from glucose data, meal times, and physiological responses.

Hosting & Access

The project is hosted on GitHub Pages for easy access and a seamless user experience. This allows users to explore the interactive charts and data visualizations directly from the web without any installation required.
