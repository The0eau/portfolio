Stock Market Predictor (AI)

This project focuses on predicting stock market trends using machine learning techniques. It involves data preparation, model training, and data visualization to develop an AI system capable of predicting stock prices based on historical data. The models used include Random Forest and Neural Networks, applied specifically to financial data.

Key Features

  • Data Preparation: Collected and cleaned historical stock market data, including financial indicators, stock prices, and market conditions, to prepare it for machine learning analysis.
